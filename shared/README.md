# Модели API

Этот каталог содержит кроссплатформенные модели данных, сгенерированные из спецификации OpenAPI (`api/openapi.yaml`). Эти модели разработаны для работы на всех платформах, поддерживаемых Kotlin Multiplatform проектом.

## Обзор моделей

### Общие модели

- `ErrorResponse` — стандартная структура ответа с ошибкой

### Модели аутентификации

- `User` — информация о профиле пользователя
- `RegisterRequest` — данные для регистрации
- `LoginRequest` — учётные данные для входа
- `LoginResponse` — ответ при входе, включая JWT-токен

### Модели управления задачами

- `Task` — полная сущность задачи
- `TaskInput` — данные для создания или обновления задачи

### Модели заметок

- `Note` — полная сущность заметки
- `NoteInput` — данные для создания или обновления заметки

### Модели расписания

- `Lesson` — элемент расписания занятий
- `ScheduleParams` — параметры запросов расписания

### Модели групп

- `GroupInfo` — информация об учебной группе

## Использование

Модели помечены аннотацией `@Serializable` из библиотеки сериализации Kotlin, что позволяет использовать их для сериализации и десериализации в формат JSON.

```kotlin
// Пример использования
val user = User(
    id = "uuid-string",
    email = "student@example.com",
    fullName = "Имя Студента",
    groupName = "CS-101"
)

// Валидация
val validationResult = ModelValidators.validateUser(user)
```
